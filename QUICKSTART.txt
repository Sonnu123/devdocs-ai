Follow these 5 simple steps to get DevDocs AI running:

┌────────────────────────────────────────────────────────────────┐
│ STEP 1: Get FREE Gemini API Key (2 minutes)                   │
└────────────────────────────────────────────────────────────────┘
1. Open: https://makersuite.google.com/app/apikey
2. Sign in with Google
3. Click "Create API Key"
4. Copy the key (looks like: AIzaSy...)

┌────────────────────────────────────────────────────────────────┐
│ STEP 2: Install Dependencies (1 minute)                       │
└────────────────────────────────────────────────────────────────┘
Open terminal in the devdocs-ai folder and run:

pip install -r requirements.txt --break-system-packages

┌────────────────────────────────────────────────────────────────┐
│ STEP 3: Add Your API Key (30 seconds)                         │
└────────────────────────────────────────────────────────────────┘
1. Copy the template:
   cp .env.example .env

2. Open .env in any text editor:
   nano .env
   (or: code .env, vim .env, etc.)

3. Replace "your_google_gemini_api_key_here" with your actual key
4. Save and close

┌────────────────────────────────────────────────────────────────┐
│ STEP 4: Build the System (2-3 minutes)                        │
└────────────────────────────────────────────────────────────────┘
Run these two commands:

python -m scraper.scraper
python -m vectorstore.store

Wait for "BUILD COMPLETE" message.

┌────────────────────────────────────────────────────────────────┐
│ STEP 5: Run the API (instant)                                 │
└────────────────────────────────────────────────────────────────┘
python -m api.main

You'll see: "API running at: http://localhost:8000"

┌────────────────────────────────────────────────────────────────┐
│ TEST IT!                                                       │
└────────────────────────────────────────────────────────────────┘
Open your browser: http://localhost:8000

Or test with curl:
curl -X POST "http://localhost:8000/ask" \
  -H "Content-Type: application/json" \
  -d '{"question": "What is a Python decorator?"}'
